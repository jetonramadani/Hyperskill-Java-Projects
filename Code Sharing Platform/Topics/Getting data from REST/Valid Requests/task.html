Valid Requests

Here is a class representing a task:

public class Task {
    private int id;
    private String name;
    private String description;
    private boolean completed;

    // getters and setters
}

Which REST request would return the task with the id = 1 as the value from the list?

@RestController
public class TaskController {
    List<Task> taskList = List.of(
        new Task(1, "task1", "A first test task", false),
        new Task(2, "task2", "A second test task", true)
    );

    @GetMapping("/tasks/{id}")
    public Task getTasks(@PathVariable int id) {
        return taskList.get(id);
    }
}
